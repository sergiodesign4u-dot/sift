03

Canonical component (owned here)

Source-type enum

MVP values: CSV (one-off import batch) and Intercom (live connection). Later, same abstraction: Zendesk, Gong. Each value carries a label, a connect method (upload vs OAuth), a status model (batch imported vs live synced), and the PII-scrub default (on). Grounded in D3; no other source is invented.

Referenced by 3.0 rows and the Home marketing copy. CSV is a Source with a batch attribute, not a separate entity (sitemap.md, E2).

04

SEO and accessibility

  • noindex (private app), robots: noindex, follow, no schema.
  • The source list is a semantic list; status is not color-only (a word plus an icon).
  • The column-mapping step labels every control; the disconnect confirm is a focus-trapped dialog.
  • Errors recover in place (re-map, re-authorize, retry); the only true dead-end (no usable feedback anywhere) lives in Flow 1, not on a screen.
  • Every target is at least 44px.
